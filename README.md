# formatho/homebrew-tap

Homebrew tap for Formatho software packages.

## How do I install these formulae?

```bash
brew tap formatho/tap
brew install <formula>
```

Or install directly:

```bash
brew install formatho/tap/<formula>
```

## Available Formulae

| Formula | Description | Repository |
|---------|-------------|------------|
| `agent-todo` | CLI for Agent Todo Platform | [github.com/formatho/agent-todo](https://github.com/formatho/agent-todo) |

## Documentation
